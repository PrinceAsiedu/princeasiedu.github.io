Beginning My Security Engineering Journey

Today I’m starting my journey to become a security-focused software engineer. Over the next two years, I’ll be documenting my progress, sharing what I learn, and showcasing projects I build along the way.

My Learning Roadmap

I’ve created a structured learning plan that covers:

  • Computer Science Fundamentals
  • Programming Mastery (Python, Go, Bash)
  • Web Security
  • DevOps & Cloud Security
  • Network & Infrastructure Security
  • Security Operations

Each month, I’ll focus on specific areas and build projects to apply what I’ve learned.

Goals for This Month

For the first month, I’m focusing on:

  1. Strengthening my understanding of data structures and algorithms
  2. Advanced Linux usage and security concepts
  3. Setting up this blog to document my journey

Stay tuned for regular updates as I progress through my roadmap!
